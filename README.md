# DarkclamBot

  A bot that gets users ready for the darkclam drop.

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g darkclam-bot
```
edge
```bash
npm install -g https://github.com/daxxog/darkclam-bot/tarball/master
```
